# Ganbaru – Prueba técnica (Gestor de tareas)

Prueba técnica desarrollada en **PHP + MySQL + HTML/JS/CSS**.

## Funcionalidad
- Crear tareas con categorías (PHP, Javascript, CSS)
- Listar tareas sin refrescar la página (AJAX)
- Eliminar tareas sin refrescar
- Relación N:M tareas–categorías
- Consulta SQL única para obtener tareas y categorías

## Stack
- PHP (PDO)
- MySQL
- HTML / CSS / JavaScript (Fetch API)
- Servidor local (XAMPP)
